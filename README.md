## 🧠 Overview

This calculator provides estimates derived from best available data — harm-reduction tables, community reporting, and published assay work.
NOT medical advice — it does not provide a prescription or a substitute for professional guidance.

It is built as a lightweight, single-file web application with a focus on clarity, consistency, and practical usability.

---

## ⚙️ How it works

This calculator provides **dose estimates derived from the best available data**, including harm-reduction guidelines, community-reported experiences, and published potency analyses.

The model integrates:

* Body weight scaling
* Strain potency adjustments
* Structured trip-level ranges

These inputs are mapped into a unified dosing framework designed to reflect real-world variability while remaining consistent and interpretable.

Further details on methodology, assumptions, and data sources are available in app.

## 📱 Features

* Structured trip-level dosing system
* Strain potency modifiers
* Interactive dose mapping (slider-based refinement)
* Contextual safety warnings
* Offline support (PWA-ready)
* No data collection or tracking
